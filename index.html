Mes Projets

Kasa

Projet Kasa – Application Front-End Immobilière avec React Développé de zéro pour un cours, ce projet est une application front-end pour une plateforme immobilière, utilisant React et React Router pour une navigation fluide entre les pages. L'architecture en composants assure la modularité et la facilité de maintenance. SASS est employé pour gérer les styles, offrant une personnalisation avancée et un design responsive qui s'adapte parfaitement aux différents appareils, du mobile à l'ordinateur de bureau. Ce projet démontre mes compétences approfondies en développement web moderne et en conception UI/UX responsive.

Sophie Bluel Architecte D'interieur

Projet de Site Web pour Sophie Bluel, Architecte d'Intérieur Ce projet met en œuvre une galerie dynamique et des fonctionnalités d'administration en JavaScript, enrichies par des appels API pour la gestion des œuvres d'art. Il offre une interaction utilisateur avancée avec des filtres de catégorie et la capacité d'ajouter ou de supprimer des images en fonction du statut de connexion de l'utilisateur. Le site est conçu pour être réactif et accessible, utilisant HTML5, CSS3, et des modules JavaScript modernes pour une expérience utilisateur optimale.
